
## Docstring Format

"""
<Short one-line summary of what the function does.>

<Optional longer description explaining behavior, validation,
and important details about how the function works.>

Args:
    <param_name> (<type>):
        <Description of the parameter>

        Examples:
            <example 1>
            <example 2>

Returns:
    <type>:
        <description of return value>
        <possible return states if any>

Raises:
    <ExceptionType>:
        <when this exception occurs>

    None:
        <if all errors are handled internally>

Features:
    - <feature 1>
    - <feature 2>
    - <feature 3>

Example:
    >>> <function_call_example>
    <expected_output>
"""
